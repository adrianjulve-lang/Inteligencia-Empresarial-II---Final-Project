## Contexto del Proyecto
- **Empresa:** 50 Doctors México — cadena hospitalaria de lujo a precio accesible
- **Reto:** evaluar expansión/alianzas con aseguradoras españolas (Sanitas, Adeslas, Asisa)
- **Curso:** Inteligencia Empresarial II, UFV, Prof. César Moreno Pascual PhD

## Frameworks clave (ver docs/)
- **AMC Framework** (Chen & Miller): Awareness → Motivation → Capability
- **Shell Method**: Driving Forces → Uncertainties → Matrix → Narratives → Signposts
- **Exchange Value**: Valor = Valor de Referencia + Valor Diferencial
- **ECOMO NPV**: evaluación económica de escenarios
- **ABP**: Assumption-Based Planning (Load-bearing / Vulnerable / Robust)
- **KITs**: Key Intelligence Topics
