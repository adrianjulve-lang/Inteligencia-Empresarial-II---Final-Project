# Inteligencia Empresarial II — Proyecto Final
**Universidad Francisco de Vitoria | Profesor: César Moreno Pascual PhD**

## La Empresa: 50 Doctors (Fifty Doctors) México
Cadena hospitalaria mexicana de mayor crecimiento. Modelo: hospitales de lujo a precios accesibles.
- Fundadores: Javier de Lope Francés (presidente), Jorge Marcos Martínez (DG)
- Presencia: Puebla, Torreón, La Paz BCS, Cancún + 14 hospitales en desarrollo
- Meta: 50 hospitales en México + expansión internacional

## Reto Estratégico
Evaluar si 50 Doctors puede vender servicios / establecer alianzas con aseguradoras españolas (Sanitas, Adeslas) o actores del sistema sanitario español.
